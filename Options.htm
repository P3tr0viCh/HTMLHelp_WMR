4 Настройка

Настройки содержат информацию, необходимую для правильной работы программного обеспечения. Работа с настройками требуется для максимальной адаптации программы к конкретным условиям функционирования и, в частности, к существующему программному обеспечению и оборудованию вагонных весов.

Настройки хранятся в зашифрованном файле WMR.cfg, находящемся в рабочем каталоге программы. Данный файл не предназначен для ручного редактирования каким-либо образом, например, в текстовом редакторе. В случае внесения в него изменений, удаления или другой порчи запуск программы становится невозможен, при этом выводится сообщение об ошибке. Желательно, после внесения изменений предусмотренным способом, сохранять копию файла на каком-либо защищённом ресурсе.

Доступ к настройкам осуществляется нажатием кнопки "Настройки" на главном окне. Данная кнопка доступна только пользователю "Администратор".
Для удобства, все настройки разбиты по функциональным группам, каждая из которой находится на соответствующей вкладке.

Окно настроек содержит четыре кнопки:

  • "ОК" – закрывает окно и сохраняет сделанные в настройках изменения;
  • "Отмена" – закрывает окно сбрасывая внесённые изменения;
  • "В файл..." – сохраняет настройки в текстовый файл. Имя и расположение указываются в диалоге сохранения. Пароли не сохраняются;
  • "Из файла..." – вызывает диалог открытия файла, в котором можно указать имя текстового файла с настройками. После загрузки настроек можно принять загруженные значения нажав кнопку "ОК", либо отказаться от них нажав кнопку "Отмена".
Кнопки "В файл..." и "Из файла..." предназначены для резервного сохранения и восстановления настроек.

4.1 Вкладка "Программа"


Группа "Режим работы" включает различные разрешения. Включение переключателя "Разрешить редактирование" позволяет осуществлять редактирование данных, отключение – разрешает только просматривать данные. Отключение данной опции также отключает возможность добавления данных посредством провески состава и удаление записей.
Включение "Разрешить ручной ввод" разрешает осуществлять провеску при помощи ручного управления, а включение "Разрешить автоматический ввод" – выполнять провеску в автоматическом режиме при помощи путевого датчика и весоизмерительного прибора CAS или при помощи модуля "Динамика". Включение или отключение данных опций также отображает или скрывает соответствующие кнопки в группе "Взвешивание" на главном окне.
Включение "Разрешить удаление записей" позволяет удалять строки в окне редактирования состава. В случае отключения данной опции кнопка "Удалить" в окне редактирования скрывается.
Отключение данных опций не распространяется на администратора.
Включение "Автоинкремент номера выпуска" позволяет автоматически увеличивать номер выпуска при выборе печи в окне редактирования на единицу.
В поле "Длина вагона" вносится длина вагона в метрах. Это значение используется для вычисления средней скорости прохождения состава по весам при автоматическом взвешивании. Простой способ измерить длину вагона – измерить расстояние между центрами одинаковых осей (например, крайних левых) двух соседних вагонов.
В списке "Порт индикатора" указывается COM-порт компьютера, к которому подключен весоизмерительный прибор CAS CI-6000A. В случае отсутствия у компьютера COM-портов, можно воспользоваться кабелем-преобразователем интерфейсов (переходником) USB-COM, при этом для него необходимо в диспетчере устройств Windows задать номер порта от 1 до 9.
В поле "№ вагона по умолчанию" указывается часть номера вагона, который будет автоматически вноситься в соответствующее поле окна редактирования при добавлении новой записи. При редактировании пользователю нужно будет добавить только недостающую часть номера вагона.

4.2 Вкладка "Весы"

В этой вкладке указываются данные о весовом пункте, которые отображаются на странице списка точек взвешивания Веб-интерфейса ИСВС.
В поле "Номер весов" указывается уникальный номер весового пункта. Данный номер должен быть задан во время начальной установки программы и после начала работы и сохранения данных на ИСВС не изменяться. В случае необходимости изменения, после задания нового номера нужно вручную обновить таблицы ИСВС.
Настройки данной вкладки используются только в случае включения какой-либо настройки вкладки "MySQL".

4.3 Вкладка "MySQL"

Настройки вкладки отвечают за взаимодействие с ИСВС. Включение переключателя "Сохранять брутто на сервер" позволяет выполнять сохранение провесок на ИСВС посредством встроенных в программу механизмов. Включение "Сохранять и загружать тару с сервера" позволяет сохранять вес порожних вагонов на ИСВС и использовать тару вагонов, хранящуюся на ИСВС. Включение переключателя "Сохранять и загружать номер выпуска с сервера" позволяет использовать номер выпуска, хранящийся на ИСВС.
В поле "Дополнительно отображать данные со следующих весов" вносятся через запятую номера весовых пунктов, данные с которых нужно показывать в списке провешенных вагонов и списке тар.

4.4 Вкладка "Настройки MySQL"

Здесь указывается адрес и порт ИСВС, а также имя и пароль пользователя для доступа к базе данных ИСВС.

4.5 Вкладка "Авитек"

Включение переключателя "Сохранять данные на сервер, используя модуль "Авитек"" позволяет использовать для сохранения провешенных вагонов на ИСВС модуль "Авитек".
При включении данного переключателя, нужно выключать переключатель "Сохранять брутто на сервер" вкладки "MySQL", в противном случае сохранение данных на ИСВС будет выполняться дважды.
Расположение модуля указывается в поле "Путь к модулю "Авитек"".

4.6 Вкладка "Номера печей"

Данная вкладка содержит список печей, который отображается в окне редактирования.
Изменение списка выполняется кнопками, расположенными на вкладке.
Кнопка "Добавить" позволяет добавить новую запись в список, кнопка "Изменить" изменяет выделенную в списке строку и кнопка "Удалить" – удаляет выделенные строки.
Кнопка "Добавить" доступна для нажатия всегда, а кнопки "Изменить" и "Удалить" – при наличии выделенной строки в списке.
Вместо нажатия кнопок можно использовать двойной щелчок мыши по списку. Добавление записи выполняется щелчком по пустому полю списка, изменение – щелчком по строке списка.

Нажатие кнопок "Добавить" или "Изменить" или двойной щелчок по списку вызывает появление окна, показанного ниже.

При добавлении записи поле, предназначенное для ввода, пустое. При изменении записи, оно содержит значение выделенной строки. Для внесения в список изменений нужно ввести название печи и нажать кнопку "ОК".

Работа со списком аналогична вкладке "Номера печей", при редактировании отображается окно, показанное ниже.

Работа со списком аналогична вкладке "Номера печей", при редактировании отображается окно, показанное ниже.

4.9 Вкладка "Тип прибора"

Вкладка позволяет выбирать механизм взаимодействия с весовым оборудованием.
В случае выбора модуля "Динамика" ООО "Веском", необходимо в соответствующем поле указать расположение каталога программного обеспечения ООО "Веском".

4.10 Вкладка "Распознавание вагонов"

Настройки вкладки предназначены для автоматического взвешивания с использованием весоизмерительного прибора CAS.
Программное обеспечение предназначено для использования на весовых пунктах, большая часть груза на которых провешивается в составах, сформированных определённым образом. В случае необходимости выполнить взвешивание состава другого типа, необходимо использовать ручное взвешивание.

4.11 Вкладка "Пользователи"

Данная вкладка предназначена для редактирования списка пользователей, которым разрешена работа с программой. Полученный список отображается в окне "Вход в программу".

Работа со списком аналогична вкладке "Номера печей", за исключением кнопки "Удалить", которая недоступна при выборе первой строки, так как первым в списке всегда идёт пользователь с правами администратора.
Ввод пароля не является обязательным. В случае отсутствия пароля у пользователя, для него в колонке "Пароль" выводится надпись "Нет", если пароль задан – надпись "Да".
При редактировании отображается окно, показанное ниже.
В данном окне в соответствующие поля вводятся имя пользователя и пароль. В поле "Подтверждение" нужно для контроля также ввести пароль.
