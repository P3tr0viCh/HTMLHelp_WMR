1 Общая информация

1.1 Назначение

Программное обеспечение для вагонных весов типа "МР" реализует под управлением оператора процедуру взвешивания в движении любых двух- и четырёхосных железнодорожных вагонов, процедуру взвешивания в статике любых вагонов и редактирование базы данных взвешенных вагонов.

1.2 Интерфейс

Интерфейс реализован в виде системы диалоговых окон, управление в которых осуществляется посредством клавиатуры и графического манипулятора (мыши). Диалоговые окна соответствуют технологии взвешивания и получения его результатов.

1.3 База данных

База данных представляет собой хранилище результатов взвешивания вагонов, при этом для каждого вагона сохраняются различные информационные поля: дата и время взвешивания, номер вагона, род вагона, грузоподъёмность, брутто, тара, скорость при взвешивании в движении и другие.

1.4 Основные операции

Основные операции, реализуемые программным обеспечением:
  • идентификация (распознавание) подвижного состава (локомотивов и вагонов) при взвешивании в движении;
  • взвешивание (определение массы брутто) вагона в движении и в статике;
  • расчёт массы нетто, чистого веса и потерь по массам брутто, тары "до слива" и тары "после слива";
  • сохранение результатов в локальной базе данных;
  • передача результатов на интеграционный сервер весовых систем ООО "Авитек Плюс" (далее ИСВС).

1.5 Основные результаты

Основные результаты работы программы:
  • результаты взвешивания вагонов в движении;
  • результаты взвешивания вагонов в статике.

1.6 Техническое обеспечение

Конфигурация персонального компьютера должна соответствовать требованиям версии операционной системы, установленной на нём.
Для взаимодействия с весами необходим весоизмерительный прибор CAS CI-6000A, либо, при использовании на вагонных весах производства ООО "Веском", соответствующее этим весам оборудование.

1.7 Программное обеспечение

Операционная система Microsoft Windows 2000, Microsoft Windows XP или Microsoft Windows 7.
В случае использования ИСВС: драйвер MySQL ODBC версии 3.51.11, в дополнение к нему может применяться модуль обмена ООО "Авитек Плюс" версии 1.7.
При использовании на вагонных весах производства ООО "Веском" должно быть установлено соответствующее программное обеспечение – модуль "Динамика" ООО "Веском" версии 7.1.3.0.
