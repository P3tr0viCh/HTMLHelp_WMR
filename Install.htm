3 Установка

3.1 Установка программного обеспечения

Установка программы предполагает наличие предварительно установленной операционной системы Microsoft Windows, указанной в п. 1.7. Установка проводится при условии входа в операционную систему с правами Администратора.

Список основных файлов программного обеспечения:

  • WMR.exe – исполняемый файл программы;
  • WMR.mdb – локальная база данных в формате Microsoft Access 2000;
  • WMR.cfg – основной конфигурационный файл;
  • Help.chm – справочная система;
  • WMR.ini – дополнительный конфигурационный файл, содержащий информацию о положениях диалоговых окон на рабочем столе, ширине колонок и т.п. Необязательный файл, в случае отсутствия создаётся автоматически;
  • WMR.log – протокол работы, находится в подпапке "log". Необязательный файл, в случае отсутствия создаётся автоматически.
Данные файлы необходимо скопировать в любую папку, обеспечив пользователю операционной системы соответствующие права доступа на запись в эту папку.

3.2 Права доступа в файловой системе NTFS

Для нормальной работы программного обеспечения необходимо обеспечить для всех пользователей операционной системы доступ к файлам программы, расположенным в рабочей папке (по умолчанию она называется WMR). Для этого необходимо воспользоваться вкладкой "Безопасность" окна свойств рабочей папки и убедиться в том, что пользователям "Пользователи" установлены разрешения изменения, чтения и выполнения, чтения и записи.

3.3 Установка драйвера MySQL ODBC

Для обеспечения возможности сохранения результатов взвешивания на ИСВС необходимо установить драйвер ODBC. Данная версия программного обеспечения использует драйвер MySQL ODBC версии 3.51.11.
Установочный комплект содержится в файле "MyODBC 3.51.11.exe". Для установки драйвера необходимо иметь права Администратора.
